# Basic Speech Recognition

This project is a Python-based speech recognition system that records voice input
from the microphone and converts it into text using Google Speech Recognition API.

## Features
- Records audio using microphone
- Saves audio in WAV format
- Converts speech to text
- Displays recognized text as output

## Technologies Used
- Python
- SpeechRecognition
- SoundDevice
- NumPy
- SciPy

## How to Run the Project
1. Install required Python libraries
2. Run main.py file
3. Speak into the microphone
4. View the recognized text in terminal
